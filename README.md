# Sales Forecasting & Demand Intelligence Dashboard

## Project Overview

This project analyzes retail sales data using Machine Learning and Time Series Forecasting techniques.

## Features

- Sales Overview Dashboard
- Monthly Sales Trend
- Regional & Category Analysis
- Sales Forecasting using XGBoost
- Isolation Forest Anomaly Detection
- Product Demand Segmentation using K-Means

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Scikit-learn
- XGBoost

## Dataset

Superstore Sales Dataset
